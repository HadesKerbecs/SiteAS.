TAG: PRE

Utilização da TAG PRE




Esta TAG é utilizada para representar o texto pré-formatado, vai ser escrito do mesmo modelo que o código fonte.
Se deseja ver o conteudo do jeito que foi editado.

O link abaixo ira direcionar a uma página com explicações completas sobre a TAG em si.

Exemplo: Foi posto o seguinte código CSS, no código em si, existe a TAG PRE, o Resultado está abaixo.

          color:blue;
        }
